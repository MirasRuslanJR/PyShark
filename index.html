Основные Модули
  • pybricks.hubs - работа с хабами
  • pybricks.pupdevices - моторы и датчики
  • pybricks.parameters - порты и направления
  • pybricks.tools - вспомогательные функции
  • pybricks.robotics - DriveBase для роботов
Моторы

Создание мотора:

motor = Motor(Port.A)

Основные команды:

  • motor.run(500) - запустить с постоянной скоростью
  • motor.run_angle(500, 360) - повернуть на угол
  • motor.run_time(500, 2000) - работать заданное время
  • motor.stop() - остановить мотор
Датчики

Датчик цвета:

color = ColorSensor(Port.B)
color.color() # получить цвет

Ультразвуковой датчик:

ultrasonic = UltrasonicSensor(Port.C)
ultrasonic.distance() # расстояние в мм

Датчик силы:

force = ForceSensor(Port.D)
force.force() # сила в Н
DriveBase

Создание робота:

robot = DriveBase(left_motor, right_motor, wheel_diameter, axle_track)

Команды движения:

  • robot.straight(distance) - прямо
  • robot.turn(angle) - поворот
  • robot.curve(radius, angle) - дуга
  • robot.drive(speed, turn_rate) - свободное движение
Время и Ожидание

Функции времени:

from pybricks.tools import wait

wait(1000) # ждать 1 секунду

StopWatch для измерения времени:

watch = StopWatch()
watch.time() # текущее время в мс
watch.reset() # сбросить
Полезные Примеры

Движение до препятствия:

while ultrasonic.distance() > 100:
robot.drive(200, 0)
robot.stop()

Следование за линией:

while True:
if color.reflection() > 50:
robot.drive(100, 20)
else:
robot.drive(100, -20)

Советы для Начинающих
  • Всегда проверяй правильность подключения портов
  • Начинай с малых скоростей (200-300) при тестировании
  • Используй wait() между командами для плавности
  • Калибруй датчики перед началом работы
  • Сохраняй код часто и делай резервные копии
  • Экспериментируй с параметрами для лучших результатов
